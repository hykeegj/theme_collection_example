# 웹페이지 TOP/BOTTOM 버튼 기능 구현

## TOP/BOTTOM 버튼 기능

---

### TOP/BOTTOM 버튼이 표시되는 조건

-   화면 최초 로드 시
-   스크롤 시
-   TOP/BOTTOM 버튼 클릭 시

---

### TOP/BOTTOM 버튼이 사라지는 조건

-   버튼 노출 후 2초가 지났을 시 2초에 걸쳐 서서히 사라짐
